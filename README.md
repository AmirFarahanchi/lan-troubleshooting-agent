# LAN Troubleshooting Agent

An AI-assisted troubleshooting agent designed to diagnose and resolve common LAN issues in MikroTik and Windows environments.

## What this agent does
- Systematically diagnoses LAN connectivity and performance problems
- Works with MikroTik RouterOS and Windows-based networks
- Follows real-world network troubleshooting methodology
- Produces clear, actionable steps (not generic advice)

## Use cases
- No internet access on LAN
- Slow or unstable network
- DHCP or IP conflicts
- Gateway or DNS misconfiguration
- MikroTik misrouting or firewall issues

## How to use
1. Fill out the intake form (`intake_form.md`)
2. Apply the agent logic defined in `agent_prompt.txt`
3. Follow the diagnostic and resolution steps provided

## Target audience
- Network technicians
- System administrators
- IT support engineers
- Students learning real-world LAN troubleshooting
